# Enable repository analysis

## 1) Turn on Advanced Security

In GitHub:

- **Settings** → **Advanced Security**
- Enable:
  - Secret Protection
  - Code Security
  - Dependency Graph

![alt text](image-1.png)

## 2) Turn on Code Quality

In GitHub:

- **Settings** → **Code Quality**

![alt text](image-2.png)

## 3) Wait for results

It takes a few minutes to populate findings.
